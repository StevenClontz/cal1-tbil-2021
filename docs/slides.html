Definition 1.6.3

A function has a vertical asymptote at \(x=a\) when or The limit being equal to positive infinity means that we can make the output of \(f(x)\) as large as we want as long as we are sufficiently close to \(x=a\text{.}\) Similarly, the limit being equal to negative infinity means that we can make the output of \(f(x)\) as small as we want as long as we are sufficiently close to \(x=a\text{.}\)
