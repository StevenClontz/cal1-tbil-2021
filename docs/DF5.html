Section 2.5 The chain rule (DF5)

Learning Outcomes

  • Compute derivatives using the Chain Rule.

Activity 2.5.1.
(a)

Suppose that the function \(f(x) = -x^2+5 \) and \(g(x)=2x-1\text{.}\) Which of the following is a formula for \(f(g(x))\text{?}\)

  1. \(\displaystyle -4x^2 +4x +4 \)

  2. \(\displaystyle 4x^2 -4x +5 \)

  3. \(\displaystyle -2x^2 +9 \)

  4. \(\displaystyle -2x^2 +4 \)

(b)

One of the options above is a formula for \(g(f(x))\text{.}\) Which one?

Activity 2.5.2.
(a)

Consider the composite function \(\sqrt{e^x}\text{.}\) Which function is the outside function \(f(x)\) and which one is the inside function \(g(x)\text{?}\)

  1. \(\displaystyle f(x)=x^2 \, , \, g(x)=e^x \)

  2. \(\displaystyle f(x)=\sqrt{x} \, , \, g(x)=e^x \)

  3. \(\displaystyle f(x)=e^x \, , \, g(x)=\sqrt{x} \)

  4. \(\displaystyle f(x)=e^x \, , \, g(x)=x^2 \)

(b)

Using properties of the exponents, we can rewrite the original function as \(e^{\frac{x}{2}}\text{.}\)
