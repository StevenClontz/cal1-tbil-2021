Section 3.7 Graphing with derivatives (AD7)

Learning Outcomes

  • Sketch the graph of a differentiable function whose derivatives satisfy given criteria.

In Section 3.5 and Section 3.6 we learned how the first and second derivatives give us information about the graph of a function. Specifically, we can determine the intervals where a function is increasing, decreasing, concave up, or concave down as well as any relative extrema or inflection points. Now we will put that information together to draw a sketch of the graph.

Activity 3.7.5.

For each prompt that follows, sketch a possible graph of a function on the interval \(-3 \lt x \lt 3\) that satisfies the stated properties.

(a)

\(y = f(x)\) such that \(f\) is increasing on \(-3 \lt x \lt 3\text{,}\) concave up on \(-3 \lt x \lt 0\text{,}\) and concave down on \(0 \lt x \lt 3\text{.}\)

(b)

\(y = g(x)\) such that \(g\) is increasing on \(-3 \lt x \lt 3\text{,}\) concave down on \(-3 \lt x \lt 0\text{,}\) and concave up on \(0 \lt x \lt 3\text{.}\)

(c)

\(y = h(x)\) such that \(h\) is decreasing on \(-3 \lt x \lt 3\text{,}\) concave up on \(-3 \lt x \lt -1\text{,}\) neither concave up nor concave down on \(-1 \lt x \lt 1\text{,}\) and concave down on \(1 \lt x \lt 3\text{.}\)

(d)

\(y = p(x)\) such that \(p\) is decreasing and concave down on \(-3 \lt x \lt 0\) and is increasing and concave down on \(0 \lt x \lt 3\text{.}\)

We must also be mindful of other characteristics of a function, such as the domain and the existence (or non-existence) of horizontal and vertical asymptotes. Activity 3.7.6 includes those aspects in addition to increasing, decreasing, and concavity.

Assume that \(f(x)\) has vertical asymptotes at each \(x\)-value where \(f(x)\) does not exist, that \(\lim_{x\to-\infty}f(x)= 1\text{,}\) and that \(\lim_{x\to\infty}f(x)= -1\text{.}\)

(c)

Now we will practice sketching the graph of a function from start to finish. We'll begin with an overview of the process.

Remark 3.7.7. A guide to curve sketching.

  1. Identify the domain of the function.

  2. Identify any vertical or horizontal asymptotes, if they exist.

  3. Find \(f'(x)\text{.}\) Then use it to determine the intervals where the function is increasing and the intervals where the function is decreasing. State any relative extrema.

  4. Find \(f''(x)\text{.}\) Then use it to determine the intervals where the function is concave up and the intervals where the function is concave down. State any inflection points.

  5. Put everything together and draw sketch.
