Section 2.3 Elementary derivative rules (DF3)

Learning Outcomes

  • Compute basic derivatives using algebraic rules.

Observation 2.3.1.
If we recall from the previous section we learned about how to find the derivative of functions using the limit definition of a derivative. From the activities we see that this gets cumbursome when the functions are more complicated. In this section we will discuss a more convenient way to to calculate derivatives.
Activity 2.3.2.

In this activity we will try to deduce what the derivative of a power function. Note, a power function is one in which the function is of the form \(f(x) = x^{n}\) where \(n = 1, 2, 3,...\text{.}\)

(a)

Using the limit defintion of a function what is \(f'(x)\) for the power function \(f(x) = x\text{?}\)

  1. -1

  2. 1

  3. 0

  4. Does not exist

(b)

Using the limit defintion of a function what is \(f'(x)\) for the power function \(f(x) = x^{2}\text{?}\)

  1. 0

  2. \(\displaystyle -2x\)

  3. \(\displaystyle 2x\)

  4. \(\displaystyle 2x+1\)

(c)

Using the limit defintion of a function what is \(f'(x)\) for the power function \(f(x) = x^{3}\text{?}\)

  1. \(\displaystyle 3x^2\)

  2. \(\displaystyle -3x^2\)

  3. \(\displaystyle 3x^2-3x\)

  4. \(\displaystyle -3x^2+3x\)

(d)

WITHOUT using the limit defintion of a function what is \(f'(x)\) for the power function \(f(x) = x^{4}\text{?}\) (See if you can find a pattern from the first three tasks of this activity.)

  1. \(\displaystyle 3x^2\)

  2. \(\displaystyle 3x^3\)

  3. \(\displaystyle 4x^2\)

  4. \(\displaystyle 4x^3\)

Definition 2.3.3.

The derivative of the function\(f(x) = x^n\text{,}\) for any nonzero real number \(n\)is

\begin{equation*} f'(x) = nx^{n-1}. \end{equation*}
