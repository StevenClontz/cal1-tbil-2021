Section 1.2 Limits numerically (LT2)

Learning Outcomes

  • Infer the value of a limit based on nearby values of the function.

Activity 1.2.1.

Based on the values of Table 1.2.2, what is the best approximation for \(\lim_{x\to 7} f(x)\text{?}\)

  1. the limit is approximately 7

  2. the limit is approximately 0.17

  3. the limit is approximately 0.16

  4. the limit is approximately 0.1667

  5. the limit is approximately 6.9999

Remark 1.2.3.

Notice that the value we obtained in Activity 1.2.1 is only an approximation, based on the trends that we have seen within the table.

Activity 1.2.4.

In Activity 1.1.1 we got an approximation to the limit of the function as \(x\) tends to 2. Now let us say you are given the same plot as in Activity 1.1.1, but also given a table of numerical values (Table 1.2.5) for the function. Given this new information which of the choices below best describes the limit of the function as \(x\) tends to 2?

  1. There is not enough information because we do not know the value of the function at \(x = 2\)

  2. Can be approximated to be 1 because the data in the table and the graph show that from the left and the right the function approaches 1 as \(x\) goes to 2

  3. Can be approximated to be 1 because the values appear to approach 1 and the graph appears to approach 1, but we should zoom in on the graph to be sure

  4. Cannot be approximated because the function might not exist at \(x = 2\text{.}\)

Activity 1.2.6.

Based on Table 1.2.7, what information can be inferred about \(\lim_{x\to 1^-}f(x)\text{,}\) \(\lim_{x\to 1^+}f(x)\text{,}\) and \(\lim_{x\to 1}f(x)\text{?}\)

  1. \(\lim_{x\to 1^-}f(x) = -0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = 0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x) = 0\)

  2. \(\lim_{x\to 1^-}f(x) = -0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = 0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x)\) does not exist

  3. \(\lim_{x\rightarrow 1^-}f(x) = 0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = -0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x)\) does not exist

  4. \(\lim_{x\rightarrow 1^-}f(x) = 0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = -0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x) = 0\)

Activity 1.2.8.

Consider the following function \(f(x)= 3x^3 + 2x^2- 5x+20.\)

(a)

Of the following options, which of the 5 values would be best to help you find \(\lim_{x \to 2}f(x)\) numerically?

  1. 1.9, 1.99, 2.0, 2.01, 2.1

  2. 1.98, 1.99, 2.0, 2.01, 2.02

  3. 1.8, 1.9, 2.0, 2.1, 2.2

  4. 1.0, 1.5, 2.0, 2.5, 3.0

(b)

Use the values that you chose in part A to calculate an approximation for \(\lim_{x \to 2}f(x)\text{.}\)

(c)

Which value best describes the limit that you obtained in part (b)?

  1. the approximate value is 41.25

  2. the approximate value is 41.5

  3. approximate value is 41.75

  4. approximate value is 42

Activity 1.2.12.

In this activity you will study the velocity of Usain Bolt in his Beijing 100 meters dash. He completed 100 meters in 9.69 seconds for an overall average speed of 100/9.69 = 10.32 meters per second (about 23 miles per hour). But this is the average velocity on the whole interval. How fast was he at different instances? What was his maximum velocity? Let's explore this. The table Table 1.2.13 shows his split times recorded every 10 meters.

(a)

What was the average velocity on the first 50 meters? On the second 50 meters?

(b)

What was the average velocity between 30 and 50 meters? Between 50 and 70 meters?

(c)

What was the average velocity between 40 and 50 meters? Between 50 and 60 meters?

(d)

What is your best estimate for the Usain's velocity at the instant when he passed the 50 meters mark? This is your estimate for the instantaneous velocity.

(e)

Using the table of values, explain why 50 meters is NOT the best guess for when the instantaneous velocity was the largest. What other point would be more reasonable?
