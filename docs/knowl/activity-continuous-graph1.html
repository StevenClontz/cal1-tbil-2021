Activity 1.4.2.

A function \(f\) defined on \(-4 \lt x \lt 4\) has the graph pictured below. Use the graph to answer each of the following questions.

  1. For each of the values \(a = -3\text{,}\) \(-2\text{,}\) \(-1\text{,}\) \(0\text{,}\) \(1\text{,}\) \(2\text{,}\) \(3\text{,}\) determine whether the limit \(\lim_{x \to a} f(x)\) exists. If the limit does not exist, be ready to explain why not.

  2. For each of the values of \(a\) from Item 1 where \(f\) has a limit, determine the value of \(f(a)\) at each such point.

  3. For each such \(a\) value, is \(f(a)\) equal to \(\lim_{x \to a} f(x)\text{?}\)

  4. Use your understanding of continuity to determine whether \(f\) is continuous at each value of \(a\text{.}\)

  5. Discuss any revisions you want to make to your definition of continuity that you arrived at toward the end of Activity 1.4.1?
