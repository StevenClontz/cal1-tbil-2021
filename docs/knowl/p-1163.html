In addition to asking whether a function is increasing or decreasing, it is also natural to inquire how a function is increasing or decreasing. Activity 3.6.1 describes three basic behaviors that an increasing function can demonstrate on an interval, as pictured in Figure 54: the function can increase more and more rapidly, it can increase at the same rate, or it can increase in a way that is slowing down. We are beginning to think about how a particular curve bends, with the natural comparison being made to lines, which don't bend at all. More than this, we want to understand how the bend in a function's graph is tied to behavior characterized by the first derivative of the function.
