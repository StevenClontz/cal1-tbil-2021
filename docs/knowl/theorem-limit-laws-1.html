Theorem 1.3.4. Limit Properties, I.

Let \(b\text{,}\) \(c\text{,}\) \(L\text{,}\) and \(K\) be real numbers, let \(n\) be a positive integer, and let \(f\) and \(g\) be functions defined on an open interval \(I\) containing \(c\) satisfying

\begin{equation*} \lim_{x \to c} f(x) = L \, \text{and} \, \lim_{x \to c} g(x) = K\text{.} \end{equation*}

Then we have the following limits as well.

  1. Constant Property \(\lim_{x \to c} b = b\)

  2. Identity Propery \(\lim_{x \to c} x = c\)

  3. Sum/Difference Property \(\lim_{x \to c} (f(x) \pm g(x)) = L \pm K\)

  4. Scalar Multiple Property \(\lim_{x \to c} b \cdot f(x) = bL\)

  5. Product Property \(\lim_{x \to c} f(x) \cdot g(x) = LK\)

  6. Quotient Property \(\lim_{x \to c} f(x) / g(x) = L/K\) (provided that \(K \ne 0\))
