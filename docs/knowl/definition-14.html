Definition 3.2.1.

The tangent line approximation (or linear approximation, or linearization) of a function \(f(x)\) at \(x=a\) is the tangent line \(L(x)\) at \(x=a\text{.}\) In formulas, \(L(x)\) is the linear function

\begin{equation*} L(x) = f'(a)(x-a) + f(a). \end{equation*}

Notice that this is obtained by writing the tangent line to \(f(x)\) at \((a,f(a))\) in point-slope form and calling the resulting linear function \(L(x)\text{.}\)
