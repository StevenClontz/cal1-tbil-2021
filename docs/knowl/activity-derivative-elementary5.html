Activity 2.3.11.

What is the derivative for the arbitrary parabola given by \(f(x) = ax^2 + bx + c\text{,}\) where \(a,\,b,\,c\) are any real numbers and \(a \neq 0\text{?}\)

  1. \(\displaystyle f'(x) = 2ax + bx + c.\)

  2. \(\displaystyle f'(x) = 2x + 1.\)

  3. \(\displaystyle f'(x) = 2ax + b.\)

  4. \(\displaystyle f'(x) = ax + b.\)
