Activity 3.9.10.

For the following limits, check if they give an indeterminate form. If they do, try to use l'Hospital's Rule. Does it help? It may or may not, or you may just need to use the rule repeatedly. Either way, try to compute the value of the following limits.

(a)

\(\displaystyle \lim_{x\to 0} \frac{\sin(x)}{x}\)

(b)

\(\displaystyle \lim_{x\to 0} \frac{e^x-1}{x}\)

(c)

\(\displaystyle \lim_{x\to \infty} \frac{3x^2+3}{x^2+ 2x}\)

(d)

\(\displaystyle \lim_{x\to 0^{+}} \frac{\ln(x)}{-x}\)

(e)

\(\displaystyle \lim_{x\to 0^{+}} \frac{\ln(x)}{1/x}\)

(f)

\(\displaystyle \lim_{x\to 0} \frac{\sin^2(3x)}{5x^3 - 3x^2}\)
