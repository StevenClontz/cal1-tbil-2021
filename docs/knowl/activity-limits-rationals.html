Activity 1.3.11.

One of the requirements to Theorem 1.3.10 when taking the limit of a rational function \(\frac{p(x)}{q(x)}\) as \(x \to c\) was that we cannot have \(q(c)=0\text{.}\) Is it possible for \(\lim_{x \to c}\frac{p(x)}{q(x)} \) to exist if \(q(c)=0\text{?}\)

  1. No, because \(\frac{p(x)}{q(x)}\) is not defnied at \(x=c\) since \(q(c)=0\text{.}\)

  2. Yes, because if you graph \(f(x)=\frac{x^2-1}{x-1}\text{,}\) the value \(f(1)\) is not defined, but the graph shows that the limit of \(f(x)\) does exist as \(x \to 1\text{.}\)

  3. No, because if you graph \(g(x)=\frac{x^2+1}{x-1}\text{,}\) the value \(g(1)\) is not defined and the graph shows that the limit of \(\lim_{x \to c}g(x)\) does not exist.

  4. Yes, because Theorem 1.3.10 does not say anything about the existence of \(\lim_{x \to c}\frac{p(x)}{q(x)}\) when \(q(c)=0\text{.}\)
