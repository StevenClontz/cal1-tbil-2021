Activity 1.2.4.

In Activity 1.1.1 we got an approximation to the limit of the function as \(x\) tends to 2. Now let us say you are given the same plot as in Activity 1.1.1, but also given a table of numerical values (Table 1.2.5) for the function. Given this new information which of the choices below best describes the limit of the function as \(x\) tends to 2?

  1. There is not enough information because we do not know the value of the function at \(x = 2\)

  2. Can be approximated to be 1 because the data in the table and the graph show that from the left and the right the function approaches 1 as \(x\) goes to 2

  3. Can be approximated to be 1 because the values appear to approach 1 and the graph appears to approach 1, but we should zoom in on the graph to be sure

  4. Cannot be approximated because the function might not exist at \(x = 2\text{.}\)
