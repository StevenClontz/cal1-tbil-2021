  1. \(y - 2 = m (x-2)\) using point-slope form with the point \((2,-2)\text{.}\)

  2. \(y + 2 = m (x-2)\) using point-slope form with the point \((2,-2)\text{.}\)

  3. \(y - 2 = m (x+2)\) using point-slope form with the point \((2,-2)\text{.}\)

  4. \(y + 2 = m (x+2)\) using point-slope form with the point \((2,-2)\text{.}\)
