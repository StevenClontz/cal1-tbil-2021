Activity 1.4.13.

Answer the questions below. It may be helpful to look at some graphs.

  1. Which values of \(c\text{,}\) if any, could make the following function continuous on the real line?

    \begin{equation*} g(x) = \begin{cases} x + c \amp x \leq 2 \\ x^2 \amp x \gt 2 \end{cases} \end{equation*}

  2. Which values of \(c\text{,}\) if any, could make the following function continuous on the real line?

    \begin{equation*} h(x) = \begin{cases} 4 \amp x \leq c \\ x^2 \amp x \gt c \end{cases} \end{equation*}

  3. Which values of \(c\text{,}\) if any, could make the following function continuous on the real line?

    \begin{equation*} k(x) = \begin{cases} x \amp x \leq c \\ x^2 \amp x \gt c \end{cases} \end{equation*}
