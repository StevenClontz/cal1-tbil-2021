  1. If \(f'(x) > 0\) and \(f''(x) > 0 \) on \((a,b)\text{,}\) then \(f\) is increasing and concave up on \((a,b)\text{.}\)

  2. If \(f'(x) > 0\) and \(f''(x) < 0 \) on \((a,b)\text{,}\) then \(f\) is increasing and concave down on \((a,b)\text{.}\)

  3. If \(f'(x) < 0\) and \(f''(x) > 0 \) on \((a,b)\text{,}\) then \(f\) is decreasing and concave up on \((a,b)\text{.}\)

  4. If \(f'(x) < 0\) and \(f''(x) < 0 \) on \((a,b)\text{,}\) then \(f\) is decreasing and concave down on \((a,b)\text{.}\)
