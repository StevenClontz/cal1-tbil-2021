Remark 2.6.1.

We've learned a lot of rules for derivatives - constant multiple, sum, difference, power, trig, product, quotient, and chain. Now we need to be sure we can take the derivative of a function that might have some of these situations mixed together.

Whenever you are given a function and asked to find its derivative, you should first analyze what's going on in that function. Questions you might ask yourself include (but are not limited to):

  1. What's the big picture? What kind of function is this?

  2. Where do you start?

  3. Is there an easier or better way to write the function?

  4. Are there products or quotients involved?

  5. Is it a composition of two (or more) functions?

  6. Is there a chain rule involved? If so, what are the outside and inside functions?

  7. What types of derivatives rules will I have to use along the way?
