Activity 1.1.17.

Sketch the graph of a function \(f(x)\) that meets all of the following criteria. Be sure to scale your axes and label any important features of your graph.

  1. \(\lim_{x\to 5^-} f(x)\) is finite, but \(\lim_{x\to 5^+} f(x)\) is infinite.

  2. \(\lim_{x\to -3} f(x)=-4\text{,}\) but \(f(-3)=0\text{.}\)

  3. \(\lim_{x\to -1^-} f(x)=-1\) but \(\lim_{x\to -1^+} f(x)\neq-1\text{.}\)
