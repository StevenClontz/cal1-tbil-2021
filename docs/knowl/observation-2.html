Observation 2.1.3.

We want to be able to consider intervals before and after \(t=2\text{.}\) A positive value of \(h\) will give an interval after \(t=2\) for example, the interval \([2,3]\) for \(h=1\text{.}\) A negative value of \(h\) will give an interval before \(t=2\) for example, the interval \([1,2]\) corresponds \(h=-1\text{.}\) In the formula above, it looks like the interval would be \([2,1]\text{,}\) but the standard notation in an interval is to write the smallest number first. This does not change the difference quotient because

\begin{equation*} \frac{f(2+h)-f(2)}{(2+h)-2} = \frac{f(2)-f(2+h)}{2-(2+h)}. \end{equation*}
