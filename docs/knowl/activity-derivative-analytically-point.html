Activity 2.2.1.

For the function \(f(x)=x-x^2\) use the limite definition of the derivative to compute \(f'(2)\text{.}\)

  1. \(\displaystyle f'(2)= \lim_{h\to 0} \frac{(2+h)-(2+h)^2-2+2^2}{h}=-3\)

  2. The limit \(f'(2)= \lim_{h\to 0} \frac{(2+h)-(2+h)^2-2}{h}=\lim_{h\to 0}\) simplifies algebraically to \(\lim_{h\to 0} -3-h \frac{4}{h}\) which does not exist, thus \(f'(2)\) is not defined.

  3. f'(2)= -2

  4. \(\displaystyle f'(2) =\lim_{h\to 0} \frac{(2+h)-(2+h)^2-2+2^2}{h}= 3\)
