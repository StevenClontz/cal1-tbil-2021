Activity 3.9.4.

Use the limit definition of the derivative to compute the following limits. Each limit is \(f'(a)\text{,}\) the derivative of some function \(f(x)\) at \(x=a\text{.}\) You need to determine the function and the point to then find the value of the limit: \(f'(a)\text{.}\)

(a)

Notice that \(\displaystyle \lim_{x\to 0} \frac{e^{2+x}-e^2}{x}\) is the limit definition of the derivative of \(e^x\) at \(x=2\) (where \(x\) was used for \(h\)). Given these observations, what is this limit equal to?

  1. 2

  2. \(\displaystyle e\)

  3. \(\displaystyle e^2\)

  4. The limit does not exist.

(b)

Consider \(\displaystyle \lim_{x\to 0} \frac{\ln(1+x)}{x}\text{.}\) This limit is also the limit definition of some derivative at a point. What is the value of this limit?

  1. 1

  2. 0

  3. \(\displaystyle \ln(2)\)

  4. The limit does not exist.
