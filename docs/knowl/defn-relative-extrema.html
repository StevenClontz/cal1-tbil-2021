Definition 3.4.10.

We say that \(f(c)\) is a relative maximum (or local maximum) of \(f\) provided that \(f(c)\geq f(x)\) for all \(x\) near \(c\text{.}\) We say that \(f(c)\) is a relative minimum (or local minimum) of \(f\) provided that \(f(c)\leq f(x)\) for all \(x\) near \(c\text{.}\)
